# 🚀 Ivan Danasuta — Portfolio V2

A modern, interactive developer portfolio built with **Next.js**, designed to showcase projects, experience, and technical skills in a clean and engaging way.

🌐 **Live Demo:** https://faith-game-b2b-regist.vercel.app/

---

## ✨ Features

* ⚡ Smooth scrolling experience (Lenis)
* 🎯 Fullscreen section-based layout
* 🎨 Modern UI with neon accent design
* 🧩 Interactive project modal (detail view)
* 📱 Fully responsive (desktop & mobile)
* 🎬 Animations powered by Framer Motion
* 🔗 Social & contact integration (WhatsApp, Email, Social Media)

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animation:** Framer Motion
* **Smooth Scroll:** Lenis
* **Deployment:** Vercel

## 🚀 Getting Started

Clone the project:

```
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Open http://localhost:3000 in your browser.

---

## 🌐 Deployment

This project is deployed using **Vercel**.

To deploy your own:

1. Push project to GitHub
2. Import repository to Vercel
3. Click **Deploy**
